To install Docker on Rasperry Pi:
  curl -fsSL get.docker.com -o get-docker.sh && sh get-docker.sh
  sudo systemctl enable docker
  (https://medium.freecodecamp.org/the-easy-way-to-set-up-docker-on-a-raspberry-pi-7d24ced073ef)

To build/install image:
   mvn clean install dockerfile:build
   docker save --output code-card-docker.tar jukebox-code-card-controller:latest
   scp code-card-docker.tar pi@<docker host name>:.
   ssh pi@<docker host name>
   docker load --input code-card-docker.tar
   docker container create --name code-card --publish 8080:8080 <image-id>
   docker container start <container-id>
